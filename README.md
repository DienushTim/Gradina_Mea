# Grădina Diana

Aplicație statică HTML/CSS/JS pentru inventarul grădinii.

## Funcții
- listă plante
- adăugare/editare/ștergere plante
- calendar anual
- stropiri și tratamente
- jurnal de lucrări
- export/import JSON
- export CSV

## Publicare pe GitHub Pages
1. Creează un repository nou, de exemplu `gradina-diana`.
2. Încarcă fișierele: `index.html`, `style.css`, `app.js`, `data.js`.
3. Intră în Settings → Pages.
4. La Source alege `Deploy from a branch`.
5. Branch: `main`, folder: `/root`.
6. Salvează. Linkul va apărea după 1–2 minute.

Datele introduse în aplicație sunt salvate local în browser. Pentru backup folosește Export JSON.
